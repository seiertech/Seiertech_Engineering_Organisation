# SeierTech Engineering Organisation

**Version:** 0.1.0  
**Status:** Foundation Bootstrap  
**Authority:** SeierTech Engineering Organisation

## Purpose

The SeierTech Engineering Organisation is the engineering operating system for SeierTech.

It governs how software products are analysed, designed, reviewed, approved, built, released and continuously improved.

This repository is product agnostic. Software platforms such as Commander C2 will be governed by this Engineering Organisation but are not defined directly in this root repository foundation.

## Operating Principle

Git is the authoritative source of truth.

Engineering work begins with Mission Orders, is reviewed by specialist engineering roles, is governed through engineering authorities, and is implemented only through approved build packs.
