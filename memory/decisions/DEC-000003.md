# DEC-000003 – Proposal Before Engineering Delivery Package

**Decision ID:** DEC-000003  
**Status:** Accepted  
**Date:** 2026-06-29

## Decision

A mission shall not directly produce a build pack.

A mission authorises an engineering proposal. The proposal must be reviewed through the EMS chain before it can become an approved build pack.

## Rationale

The EMS must preserve design discipline. Implementation instructions are only valid after evidence review and design approval.

## Operating Rule

Mission does not authorise build.

Mission authorises proposal generation.

Approved proposal authorises build pack generation.

Build does not authorise release.

Verification and scoring support the final merge or release instruction.

## Impact

The EMS lifecycle is updated to include proposal generation, proposal review, scoring and final merge instruction.
