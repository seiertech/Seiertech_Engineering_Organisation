## Finding
Multiple authorities (AUTH-006, AUTH-007) and personas (Security Architect, Chief Architect, both teams) referenced "the Risk Register" and "Platform Decision Register" as if they existed as real artefacts. Neither had been formally built as a conformant `REG-NNNNNN` register — only unreferenced legacy drafts existed.

## Impact
Any persona reasoning that genuinely tried to read or write to these registers during a real mission would have found nothing — a silent dead end, not an error.

## Reuse Guidance
When archiving "legacy" artefacts, check for live references before assuming dead = safe to discard. A register with a draft status can still be the only thing standing between doctrine that references it and a genuine gap — check usage, not just status labels.
